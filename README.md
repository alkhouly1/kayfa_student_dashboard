# Kayfa Student Analytics Dashboard

This project is an interactive Streamlit dashboard for analyzing student performance, attendance, engagement, concept mastery, risk levels, and intervention priorities.

## Tools Used
- Python
- Pandas
- Plotly
- Streamlit
- MongoDB Atlas

## Main Features
- Executive KPIs
- Course performance analysis
- Student risk analysis
- Concept failure analysis
- Attendance and submission behavior analysis
- Student segmentation
- Final recommendations

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
